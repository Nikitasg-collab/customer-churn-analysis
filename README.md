# Customer Churn Analysis

## Project Overview
In this project, I analyzed customer churn in a fintech company to understand why customers are leaving and how it impacts business growth.

## Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard)

## Approach
- Cleaned the dataset using Excel
- Analyzed churn patterns and customer behavior
- Performed segment analysis based on age, activity, products, and geography
- Used SQL to validate insights and calculate churn percentages
- Built a Power BI dashboard to visualize key findings

## Key Insights
- Around 20% of customers are churning
- Inactive customers have higher churn
- Customers using only one product are high risk
- Customers aged 30–50 show higher churn
- Germany and France have higher churn mainly due to inactive users

## Outcome
Identified high-risk customer segments and suggested strategies to improve customer engagement and reduce churn.
